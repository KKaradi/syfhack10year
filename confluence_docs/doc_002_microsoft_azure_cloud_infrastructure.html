Microsoft Azure Cloud Infrastructure

Document Type: Cloud Infrastructure

Last Updated: 2024-01-18

Owner: Cloud Architecture Team

Overview

Documentation for Microsoft Azure cloud services, including compute, storage, networking, and security configurations.

Resources and Tools

  • Azure Active Directory

    Type: service

    Description: Identity and access management service for Azure and Microsoft 365

    Access Requirements:

    • Azure AD license
    • Global administrator role

    API Endpoints:

    • https://graph.microsoft.com/v1.0/users
    • https://graph.microsoft.com/v1.0/groups

    Dependencies: Microsoft 365 subscription

    Documentation: https://docs.microsoft.com/azure/active-directory

  • Azure SQL Database

    Type: database

    Description: Managed SQL database service in Azure cloud

    Access Requirements:

    • Azure subscription
    • SQL Server authentication
    • Firewall rules

    API Endpoints:

    • https://management.azure.com/subscriptions/{id}/providers/Microsoft.Sql

    Dependencies: Azure Resource Group, Virtual Network

    Documentation: https://docs.microsoft.com/azure/sql-database

  • Azure Key Vault

    Type: service

    Description: Secure storage for keys, secrets, and certificates

    Access Requirements:

    • Key Vault access policy
    • Azure AD authentication

    API Endpoints:

    • https://{vault-name}.vault.azure.net/secrets
    • https://{vault-name}.vault.azure.net/keys

    Dependencies: Azure Active Directory

    Documentation: https://docs.microsoft.com/azure/key-vault

Databases

  • Azure SQL Production

    Type: SQL Database

    Description: Production SQL database for application data

    Connection String: Server=company-prod.database.windows.net;Database=AppDB;

    Access: SQL authentication, Azure AD integrated

Services

  • Azure App Service

    Description: Web application hosting service

    Environment: Production

    Endpoints:

    • https://company-app.azurewebsites.net
    • https://company-api.azurewebsites.net
