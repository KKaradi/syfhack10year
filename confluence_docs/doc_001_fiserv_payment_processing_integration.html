Fiserv Payment Processing Integration

Document Type: Financial Services

Last Updated: 2024-01-10

Owner: Payment Processing Team

Overview

Integration documentation for Fiserv payment processing systems, including credit card processing, ACH transfers, and fraud detection.

Resources and Tools

  • Fiserv Payment Gateway

    Type: service

    Description: Primary payment processing gateway for credit card and ACH transactions

    Access Requirements:

    • Fiserv merchant account
    • PCI DSS certification
    • API key

    API Endpoints:

    • https://api.fiserv.com/payments/v1/charges
    • https://api.fiserv.com/payments/v1/refunds

    Dependencies: SSL certificate, Merchant bank account

    Documentation: https://developer.fiserv.com/product/FirstAPI

  • Fiserv Fraud Detection

    Type: service

    Description: Real-time fraud detection and prevention service

    Access Requirements:

    • Fraud protection subscription
    • API credentials

    API Endpoints:

    • https://api.fiserv.com/fraud/v1/score
    • https://api.fiserv.com/fraud/v1/report

    Dependencies: Payment Gateway

    Documentation: https://developer.fiserv.com/fraud

Databases

  • Fiserv Transaction Database

    Type: Transaction Database

    Description: Database storing payment transaction records and audit trails

    Connection String: fiserv://secure.fiserv.com/transactions

    Access: Transaction viewer role, PCI compliance
